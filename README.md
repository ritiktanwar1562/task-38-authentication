# JWT Authentication Project

## About Project

I made this project using React.js, Express.js and JWT.

This project is about user authentication. A user can register with a username and password, login to the application and access a protected route using a JWT token.

---

## Features

- User Registration
- User Login
- JWT Authentication
- Protected Route
- Token stored in Local Storage

---

## Technologies Used

- React.js
- Express.js
- Node.js
- JWT (jsonwebtoken)
- Axios
- CORS
